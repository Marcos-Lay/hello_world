RFID传感器接口

添加传感器

  • 概述：此接口用来添加传感器
  • 接口定义：接口定义：http：// [域名]/app/rfidreader/[version]/addDevice
  • 请求方式：POST
  • 输入格式：
参数 类型 说明 是否必填
name String 传感器名称 是
sno String 传感器序列号(长度12位) 是
lightPole String 对应灯杆ID 是
  • 输出格式：
参数 类型 说明
code Integer 状态码
message String 返回提示
status String 返回状态信息
data Json 返回数据
id String 读卡器ID
sno String 读卡器序列号
name String 读卡器名称
lightPole String 读卡器对应灯杆ID
  • 输出json格式
{
    "code":200,
    "data":[
        {
            "name":"test4",
            "lightPole":"5b9374f2bb0b2049d0f12089",
            "id":"5d1c9d70631db31810910384",
            "sno":"123123123128"
        }
    ],
    "message":"OK",
    "status":"SUCCESS"
}
