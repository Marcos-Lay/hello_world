### 分页查询定时亮度任务
- 概述：此接口用来查询定时亮度任务
- 接口定义：http://[域名]/app/led2/timetask_query 
- 请求方式：POST
- 输入格式：  

| 参数 | 类型 | 说明 | 是否必填 | 是否唯一 |
| ------ | ------ | ------ | ------ | ------ |
| id | String | 16位随机字符串（唯一） | 是 | 是 |
| skip | String | 跳过条数 | 是 |
| limit | String | 查询条数（单页小于等于100条） | 是 |
